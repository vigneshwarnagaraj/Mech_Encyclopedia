How to use this manual

This is a structured engineering reference manual, not a collection of formulas pasted without context. Each section follows the pattern: concept → equation → symbol definitions → assumptions → practical interpretation.

Use it for simulation setup, quick formula lookup, pre-interview revision, or as a sanity check when a solver returns something that does not feel right.

Tip: Use the search bar to filter chapters by keyword. Formulas re-render via MathJax after every page load. Hash-based navigation means you can bookmark any page directly.

Textbook-style equations

All formulas render as proper mathematical notation with full symbol definitions, SI units, and assumptions clearly stated.

Constants & reference data

Physical and engineering constants are embedded in every relevant section and collected on the dedicated reference page (00A).

ANSYS workflow guide

Workbench, Fluent, meshing options, and post-processing — explained with what each setting does and when to use it.

Interactive calculators

Reynolds number, pipe pressure drop, beam stress, torsion, buckling, von Mises, natural frequency, and more.

Checklists & guides

Pre-run and post-run CFD and FEA checklists. Validation guides. Model selection tables. Interview preparation.

Dark & Light mode

Toggle with the moon/sun button in the sidebar or mobile bar. Preference is saved across sessions.

Navigation tips

  • Click any sidebar item to load its page — the URL hash updates so you can bookmark or share it.
  • Use the search bar to filter sidebar items by keyword.
  • Use ↑ ↓ arrow keys to scroll the sidebar; Home / End jump to top/bottom.
  • The browser back/forward buttons work with the hash router.
  • On mobile, tap ☰ Menu to open the sidebar.
GitHub Pages deployment: This site works as a static site hosted on GitHub Pages. All pages are plain HTML partials loaded via fetch(). No build step, no Node, no bundler required.
