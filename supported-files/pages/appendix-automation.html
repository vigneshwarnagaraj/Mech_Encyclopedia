Export, Automation & Python Workflows

How to export simulation data, process it with Python and pandas, and build reusable reporting workflows.

Fluent data export workflow

  1. Surface monitors: Reports → Surface Integrals → select surface and quantity (area-weighted average, mass-flow average, total flux) → enable "Write to File" before running. Produces .out file with iteration history.
  2. XY plot data: Plots → XY Plot → select axes (position, pressure, velocity component) and surface → Write → saves .csv.
  3. Residual history: Monitors → Residuals → Write → residuals.csv with all residuals vs. iteration number.
  4. Force report: Reports → Forces → select wall zones → specify direction vector → Write → forces.csv with pressure and viscous components.
  5. Full ASCII surface export: File → Export → ASCII → select surface and variables → .csv for external Python analysis.
  6. Case summary: File → Export → Case Summary — records all BCs, solver settings, and model choices in a text report.

Python: mesh independence study automation

# Load pressure drop data from three mesh levels and plot GCI import numpy as np import pandas as pd import matplotlib.pyplot as plt # Mesh study results — manually entered or loaded from CSV meshes = { 'Coarse': {'cells': 250e3, 'dp': 1425.0}, 'Medium': {'cells': 1e6, 'dp': 1381.0}, 'Fine': {'cells': 4e6, 'dp': 1364.0}, } cells = [v['cells'] for v in meshes.values()] dp = [v['dp'] for v in meshes.values()] labels = list(meshes.keys()) # GCI calculation (Richardson extrapolation) r = (cells[2]/cells[1])**(1/3) # refinement ratio p = 2 # formal order of accuracy Fs = 1.25 # safety factor GCI_med_fine = Fs * abs(dp[1]-dp[2])/dp[2] / (r**p - 1) print(f"GCI (medium→fine) = {GCI_med_fine*100:.2f}%") print(f"% change medium→fine: {abs(dp[1]-dp[2])/dp[2]*100:.2f}%") fig, ax = plt.subplots(figsize=(7, 4)) ax.semilogx(cells, dp, 'o-', color='#4da3ff', lw=2, ms=8) for i, (x, y, l) in enumerate(zip(cells, dp, labels)): ax.annotate(l, (x, y), textcoords='offset points', xytext=(0,10), ha='center', fontsize=9, color='#8fa3be') ax.set_xlabel('Mesh cell count'); ax.set_ylabel('Pressure drop (Pa)') ax.set_title('Mesh Independence Study'); ax.grid(True, alpha=0.3) plt.tight_layout(); plt.savefig('mesh_independence.png', dpi=150); plt.show()

Python: parametric study summary report

import pandas as pd import matplotlib.pyplot as plt from pathlib import Path data_dir = Path('simulation_exports/') records = [] for csv_file in sorted(data_dir.glob('*.csv')): df = pd.read_csv(csv_file) records.append({ 'Run': csv_file.stem, 'Max_Temp_C': df['T_max_K'].iloc[-1] - 273.15, 'Pressure_Drop': df['dP_Pa'].iloc[-1], 'Drag_N': df['Drag_N'].iloc[-1], 'Converged': df['residual'].iloc[-1] < 1e-4, }) summary = pd.DataFrame(records) summary.to_csv('summary_report.csv', index=False) print(summary.to_string(index=False)) fig, ax = plt.subplots(figsize=(10, 4)) ax.bar(summary['Run'], summary['Pressure_Drop'], color='#4da3ff', ec='#2a3d5a') ax.set_ylabel('Pressure Drop (Pa)'); ax.set_title('Parametric Study — ΔP by Run') plt.xticks(rotation=30, ha='right'); plt.tight_layout() plt.savefig('parametric_dp.png', dpi=150); plt.show()
