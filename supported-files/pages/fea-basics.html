FEA Formulation

The stiffness matrix, element types, shape functions, assembly, and boundary condition application.

The global FEA system

\[ [K]\{u\}=\{F\} \]
Global stiffness equation: \([K]\) = global stiffness matrix [N/m], \(\{u\}\) = nodal displacement vector [m], \(\{F\}\) = nodal force vector [N]. This is the master equation that every structural FEA solver assembles and solves. For \(n\) DOFs, this is an \(n\times n\) linear system.
\[ [k]^e=\int_{V^e}[B]^T[D][B]\,dV \]
Element stiffness matrix: \([B]\) = strain-displacement matrix (maps nodal displacements to strains), \([D]\) = constitutive matrix (maps strains to stresses). Integrated over element volume using Gauss quadrature. The global \([K]\) is assembled from all \([k]^e\) by adding contributions at shared nodes (direct stiffness method).
\[ \{F\}^e=\int_{V^e}[N]^T\vec{b}\,dV+\int_{S^e}[N]^T\vec{t}\,dS \]
Consistent nodal force vector: Body forces \(\vec{b}\) (gravity, centrifugal) and surface tractions \(\vec{t}\) (pressure, surface forces) are mapped to equivalent nodal forces using the shape function matrix \([N]\).

Shape functions and interpolation

\[ u(x)=\sum_{i=1}^n N_i(x)\,u_i = [N]\{u\}^e \]
Shape function interpolation: \(N_i(x)\) = shape function for node \(i\). Displacement at any point inside an element is interpolated from nodal displacements. Linear elements (TET4, HEXA8): straight-sided, 1st-order interpolation. Quadratic elements (TET10, HEXA20): midside nodes, parabolic interpolation — more accurate for curved boundaries and bending.
\[ [B]=\frac{\partial[N]}{\partial x}\quad\text{(strain-displacement matrix)} \]
B matrix: Differentiating the shape functions gives the strain-displacement relationship: \(\{\varepsilon\}=[B]\{u\}^e\). This is then substituted into the element stiffness integral. The accuracy of the element depends critically on how well \([B]\) represents the actual strain distribution.

Boundary conditions and constraints

\[ [K_{ff}]\{u_f\}=\{F_f\}-[K_{fc}]\{u_c\} \]
Partitioned stiffness system: Subscript \(f\) = free DOFs, \(c\) = constrained DOFs. Constrained displacements \(\{u_c\}\) are known (typically zero for fixed supports). The reduced system \([K_{ff}]\{u_f\}=\{F_f\}\) is solved for free displacements, then reactions are recovered.
Common mistakes in boundary conditions:
  • Over-constraining: too many fixed DOFs prevent physical deformation modes → unrealistically high stiffness
  • Under-constraining: rigid body modes exist → singular stiffness matrix → solver fails
  • Symmetry BCs: apply zero normal displacement AND zero rotation on the symmetry plane — both are required
  • Point constraints: create stress singularities — distribute load over an area instead

Minimum potential energy principle

\[ \Pi=U-W_{ext}=\int\frac{1}{2}\{\varepsilon\}^T\{\sigma\}\,dV-\{u\}^T\{F\} \]
Potential energy: \(U\) = strain energy stored in elastic deformation [J], \(W_{ext}\) = external work done [J]. The displacement field that minimises \(\Pi\) satisfies equilibrium. Setting \(\partial\Pi/\partial\{u\}=0\) gives \([K]\{u\}=\{F\}\) — the FEA equation.
