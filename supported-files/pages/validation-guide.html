Result Validation & Interpretation

How to decide whether your simulation result is trustworthy — before you report it.

How to validate a CFD result

  1. Check mass conservation: Inlet mass flow = outlet ± 0.1%. Use Flux Reports. Any imbalance > 0.5% indicates a leak, a BC error, or a poorly converged solution.
  2. Check convergence monitors: Residuals AND a physical quantity (drag, ΔP, T_outlet) must stabilise. Residuals alone are not convergence.
  3. Run mesh independence: Three meshes (coarse → medium → fine). If target output changes < 2–5% from medium to fine, medium is adequate. Compute GCI if a quantitative error estimate is required.
  4. Compare to theory: Pipe flow: does ΔP match Darcy-Weisbach? Flat plate: does BL thickness match Blasius? Cylinder drag: does C_D match published data at the same Re?
  5. Check velocity profiles: Laminar pipe: parabolic. Turbulent: fuller profile with thin high-gradient near-wall region. Inspect using XY plots across cross-sections.
  6. Check symmetry: Symmetric geometry + symmetric BCs → symmetric flow solution. Asymmetric result = setup error.
  7. Check y+ distribution: Must be consistent with your wall treatment choice. Plot y+ contour on all wall surfaces.
  8. Inspect streamlines near walls: Are separation and reattachment points where expected physically? Unexpected recirculation may indicate a mesh or BC error.
The most dangerous CFD result is a converged, smooth, colorful one that is wrong. Always validate before reporting.

How to validate an FEA result

  1. Check reaction forces: Sum of reaction forces = applied load. If not, the model has an error in constraints, contacts, or load application.
  2. Check deformation magnitude: Is the maximum deformation physically plausible? Compare to hand calculation: cantilever tip: δ = FL³/(3EI).
  3. Identify stress singularities: Stress at a sharp corner or point constraint that increases with mesh refinement = singularity. Add fillet radius to resolve.
  4. Mesh sensitivity at stress peaks: Refine at high-stress regions and confirm stress plateaus. If it keeps rising → singularity or insufficient resolution.
  5. Compare to analytical solution: Use beam theory, plate theory, or Lame equations for simple geometries before trusting complex FEA.
  6. Factor of safety reported correctly: FOS = σ_y / σ_vM. Report it separately — never just show the stress contour without context.
  7. Check contact status: Are contacts open/closed where physically expected? Unexpected openings indicate wrong contact type or insufficient load.

Detecting bad mesh behaviour

In CFD
  • High skewness near walls → non-physical near-wall velocities
  • Insufficient inflation layers → y+ out of range → wrong wall shear and heat transfer
  • Very high aspect ratio cells in the flow direction → excessive numerical diffusion
  • Residuals oscillate and do not converge → often poor mesh quality or a physical instability
  • Non-conformal interfaces with large size jumps → interpolation errors
In FEA
  • Stress grows without bound under refinement → singularity, not mesh convergence
  • Large aspect-ratio elements in bending → shear locking → artificially stiff
  • Linear TET4 elements in bending → overly stiff → use TET10
  • Unconnected nodes at interfaces → gap in load path → unrealistic deformation
  • Very high skewness tet elements → inaccurate Gauss point integration

When a contour plot is misleading

  • Auto-scaled colour range: If set to min/max automatically, one tiny high-stress region dominates the entire colour scale. Set manual limits to see the real distribution.
  • Exaggerated deformation scale factor: ANSYS defaults to a large scale factor (sometimes 1000×). Set "True Scale" before interpreting deflection shape.
  • Nodal vs. element-averaged stress: Averaged stresses hide jumps at element boundaries. Unaveraged stress shows raw element results — use both to assess mesh quality at interfaces.
  • Maximum at a sharp corner: May be a singularity. Add a fillet and re-check. If the peak disappears — it was artificial.
  • Streamlines vs vectors near walls: Streamlines show mean flow direction but can miss recirculation. Use velocity vectors near walls to see the actual near-wall flow direction.
  • Residual convergence plot only: Never show only the residual plot as proof of convergence. Always include a physical monitor convergence plot.
